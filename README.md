📊 Data Analysis Portfolio – Customer Behaviour Analysis

🔍 Project Overview
This project focuses on analyzing customer purchasing behavior to uncover insights that can help businesses improve decision-making, marketing strategies, and customer experience.

Using Python, SQL, and Power BI, the project explores patterns in customer demographics, spending habits, product preferences, and purchasing trends.

🎯 Objectives
Understand customer purchasing behavior
Identify high-performing categories and products
Analyze spending patterns across different demographics
Evaluate the impact of discounts and subscriptions
Provide actionable business insights through dashboards

🛠️ Tools & Technologies
Python – Data cleaning, preprocessing, and analysis
SQL – Data querying and transformation
Power BI – Interactive dashboard and visualization

📁 Dataset Description
The dataset contains customer shopping behavior information with the following features: 
Customer ID – Unique identifier for each customer
Age – Age of the customer
Gender – Customer gender (Male, Female, Other)
Item Purchased – Product purchased by the customer
Category – Product category (e.g., Electronics, Clothing)
Purchase Amount (USD) – Total amount spent per transaction
Location – Customer’s geographic location
Size – Product size (if applicable)
Color – Product color
Season – Season during which purchase was made
Review Rating – Customer rating for the purchased product
Subscription Status – Whether the customer has a subscription (Yes/No)
Shipping Type – Type of delivery (Standard, Express)
Discount Applied – Whether a discount was applied (Yes/No)
Previous Purchases – Number of past purchases by the customer
Payment Method – Mode of payment (Credit Card, Debit Card, Cash, PayPal)
Frequency of Purchases – Purchase frequency (e.g., Weekly, Monthly)
